# Social Network App

<!--### Video Demo: **URL** -->

### Setup
Clone the repository:
```bash
git clone https://github.com/petern48/social-network-app.git
cd social-network-app
```

Install dependencies:
```bash
pip install -r requirements.txt
```

Run the development server:
```bash
flask run
```

### Description:

Social Network App is a Flask App implemented to behave similar to social media platforms, such as Twitter. The programming Languages involved with developing this include Python, SQL, HTML, CSS, and Javascript.

### Features:

#### Register, Login, Logout

#### Create Post

#### Like and Comment on Posts

#### View Other User's profiles
